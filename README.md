# 🎬 Moonlight Tickets

Ứng dụng đặt vé xem phim trực tuyến xây dựng bằng MERN Stack.

## Công nghệ sử dụng

- **Frontend:** ReactJS, Styled Components, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT

## Chức năng

- Đăng ký / Đăng nhập tài khoản
- Xem danh sách & chi tiết phim
- Tìm kiếm phim
- Chọn ghế & đặt vé
- Xem vé đã đặt (có QR code)
- Đánh giá phim
- Admin: quản lý phim, rạp, người dùng

## Cài đặt & chạy dự án

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```
